# Spotify Music Analysis - February 2026 Codédex Dataset Challenge

## Overview
This project explores the Spotify Songs Dataset to analyze trends in music popularity, genres, and song characteristics. Using Python, Pandas, and data visualization libraries like Matplotlib and Seaborn, this analysis uncovers insights about which genres and artists are most popular, how song duration affects popularity, and other interesting patterns in Spotify data.

## Objective
The goal is to answer the question: 
*“Which music genres and artists have the highest popularity, and what patterns can we discover from the Spotify songs dataset?”*

## Dataset
The dataset used is the [Spotify Songs Dataset on Kaggle](https://www.kaggle.com/datasets/zaheenhamidani/ultimate-spotify-tracks-db), which contains information about thousands of songs including track name, artist, genre, popularity, duration, and more.

## Tools Used
- Python 3
- Pandas & NumPy
- Matplotlib & Seaborn
- Jupyter Notebook

## Approach
1. Load and inspect the dataset
2. Clean and preprocess the data
3. Explore trends and correlations
4. Visualize findings
5. Summarize insights
